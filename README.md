uGravity.com
====
Create simulations of Newton's law of universal gravitation. See [uGravity.com](https://ugravity.com).

Setup
---
    git clone https://github.com/sfarthin/ugravity.com ugravity.com
    cd ugravity.com
    npm install
    npm install bower -g
    bower install
    npm install -g grunt-cli
    grunt
